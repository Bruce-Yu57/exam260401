一、選擇題

1. 下列何者表示 1 到 100 中所有數的和

答案：（A）

詳解：

計算 1 到 100 中所有數的和，首項 $a_1=1$ ，末項 $a_{100}=100$ ，項數 $n=100$ 。

利用等差級數求和公式 $S_n = \frac{n(a_1+a_n)}{2}$ ，可得和為 $\frac{100 \times (1+100)}{2}$ 。

2. 已知一等差數列的首項為 -10 ，公差為 4 ，末項為 22，則關於此等差數列的敘述何者錯誤？

答案：（C）

詳解：

已知 $a_1 = -10$ ，公差 $d = 4$ ，末項 $a_n = 22$ 。

由 $a_n = a_1 + (n-1)d \Rightarrow 22 = -10 + (n-1)4 \Rightarrow 32 = 4(n-1) \Rightarrow n-1 = 8 \Rightarrow n = 9$ 。

  • （A）項數 $n = 9$ ，正確。
  • （B）$a_8 = a_1 + 7d = -10 + 7 \times 4 = -10 + 28 = 18$ ，正確。
  • （C）等差中項（即第 5 項） $= \frac{a_1 + a_n}{2} = \frac{-10 + 22}{2} = 6 \neq 12$ ，錯誤。
  • （D）總和 $S_n = \frac{9 \times (-10+22)}{2} = \frac{9 \times 12}{2} = 54$ ，正確。

4. 已知等差數列 $5, a, 9, b, c, \ldots$ ，則 $a+b+c=$ ？

答案：（D）

詳解：

數列 $5, a, 9, b, c$ 為等差數列。

$a$ 為 $5$ 與 $9$ 的等差中項 $\Rightarrow a = \frac{5+9}{2} = 7$ 。

公差 $d = 7 - 5 = 2$ 。

因此 $b = 9 + 2 = 11$ ， $c = 11 + 2 = 13$ 。

求 $a+b+c = 7 + 11 + 13 = 31$ 。

5. 若有一個等差數列的前 3 項為 $2,5,8$ ，則下列何者是此等差數列的其中一項？

  • （B）$35+1=36$（是3的倍數，當 $n=12$ 時）
  • （C）$46+1=47$（非3的倍數）
  • （D）$72+1=73$（非3的倍數）

7. 若 $a, b, c, d, e$ 為等差數列，且此五個數的和為 80 。已知 $a=6$ ，則此數列的公差為何？

8. 一等差級數 $48+45+\cdots \cdots$ 。若前 $n$ 項的和為 -105 ，則 $n$ 可能的值為何？

答案：（B）

詳解：

此等差級數首項 $a_1 = 48$ ，公差 $d = 45 - 48 = -3$ 。

前 $n$ 項和 $S_n = \frac{n[2a_1 + (n-1)d]}{2} = -105$

$\Rightarrow \frac{n[96 - 3(n-1)]}{2} = -105$

$\Rightarrow n(99 - 3n) = -210$

$\Rightarrow 33n - n^2 = -70$

$\Rightarrow n^2 - 33n - 70 = 0$

$\Rightarrow (n-35)(n+2) = 0$

解得 $n = 35$ 或 $n = -2$ （項數必為正整數，不合）。

9. 若 $x, y, 8, z, 16$ 為等比數列，則 $x$ 、 $y$ 、 $z$ 三數之積為何 ？

答案：（A）

詳解：

$x, y, 8, z, 16$ 為等比數列。

設公比為 $r$ ，則 $8 \times r^2 = 16 \Rightarrow r^2 = 2$ 。

$x$ 是 8 往回推兩項： $x = \frac{8}{r^2} = \frac{8}{2} = 4$ 。

$y \times z$ 等於等比中項 8 的平方（或首末項乘積）： $y \times z = 8^2 = 64$ 。

故 $x \times y \times z = 4 \times 64 = 256$ 。

15. 若 $(a-\mathrm{b}), 16, ~(a+3 \mathrm{b})$ 為等差數列，且 $4 a, 10, 2 b$ 也為等差數列，則 $a-\mathrm{b}=$ ？

答案：（A）

詳解：

由等差中項性質可知：

第一組數列： $2 \times 16 = (a-b) + (a+3b) \Rightarrow 32 = 2a + 2b \Rightarrow a+b = 16$ ⋯⋯①

第二組數列： $2 \times 10 = 4a + 2b \Rightarrow 20 = 4a + 2b \Rightarrow 2a+b = 10$ ⋯⋯②

② $-$ ① 得 $a = -6$ 。

將 $a = -6$ 代回①，得 $-6 + b = 16 \Rightarrow b = 22$ 。

16. 右圖是正八邊形 $A B C D E F G H$ ... 已知甲、乙第1次相遇在 $F$ 點，則第 42 次相遇在哪一點？

答案：（B）

詳解：

正八邊形共 $8$ 個邊。

甲順時針走到 $F$ 點共走了 $5$ 個邊距，乙逆時針走到 $F$ 點共走了 $3$ 個邊距。

兩人速率比為 甲：乙 $= 5 : 3$ 。

每次相遇，兩人合走了一整圈（$8$ 個邊距）。

第 $42$ 次相遇時，兩人共走了 $42 \times 8 = 336$ 個邊距。

其中甲所走的距離為 $336 \times \frac{5}{8} = 210$ 個邊距。

以甲的起點 $A$ 點算起， $210 \div 8 = 26 \dots 2$ （繞了 $26$ 圈又多 $2$ 個邊距）。

甲順時針走 $2$ 個邊距，會停在 $C$ 點。
